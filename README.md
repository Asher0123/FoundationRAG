# FoundationRAG

A modular, extensible, and production-oriented Retrieval-Augmented Generation (RAG) framework built from scratch in Python.

FoundationRAG is designed to provide clean abstractions around document ingestion, chunking, embeddings, vector storage, retrieval, and generation while remaining highly customizable and framework-agnostic.

The goal of this project is to deeply understand and implement the internals of modern RAG systems instead of relying entirely on high-level orchestration frameworks.
